# CareerFlow AI MVP 개요

## 문제 정의
취업 준비에서 가장 시간이 많이 드는 구간은  
"지원자 정보와 채용공고를 맞춰 문서를 반복 수정"하는 작업입니다.

CareerFlow AI는 이 과정을 단일 생성이 아닌 **단계형 워크플로우**로 구성해,  
분석 근거와 생성 결과를 함께 관리하는 것을 목표로 합니다.

## 제품 목표
- 이력서/포트폴리오/JD 입력을 기반으로 분석-보완-생성-면접 준비를 한 흐름으로 제공
- 결과뿐 아니라 중간 판단 근거(`WorkflowRun`, `fitAnalysisJson`)를 저장해 설명 가능성 확보
- 데모 환경에서도 저장/재방문 UX를 지원해 제품성 검증

## 핵심 가치
- **분해 가능한 AI 실행**: one-shot 프롬프트 대신 체인 기반 단계 실행
- **근거 중심 개선 루프**: 갭 탐지 -> 후속 질문 -> 재분석
- **운영 가능한 구조**: 모델 라우팅, 과호출 방지, fallback, 실행 로그

## MVP 범위
- 소스 입력/저장
- 후보자/채용공고 구조화 + 갭 분석
- 공고 대비 장·단점 스냅샷(`fitAnalysisJson`)
- 대화형 후속 보완
- 문서 3종 초안 생성 + JD 맞춤 리라이트
- 면접 대비 리포트(핵심 3 + 심화 2) 생성
- 테스트 계정 기반 저장/재조회(비인증 데모)

## 산출물 표기 원칙
- 모든 출력은 "완성본"이 아니라 **초안/도우미**로 표기
- 권장 라벨:
  - 지원동기/자기소개 초안
  - 경력기술서 초안
  - 프로젝트 근거 보강 문구
  - 면접 대비 리포트 (핵심 질문/심화 질문)
