# CareerFlow AI MVP 개요

## 목표
- 채용 지원 문서 작성 워크플로우를 AI로 구조화해, 실무형 Product Engineer 역량을 증명한다.
- 이력서/포트폴리오/채용공고 입력을 기반으로 분석-보완-생성-리라이트까지 한 흐름으로 제공한다.

## 핵심 가치
- 단일 프롬프트가 아닌 단계형 LLM 체인
- 근거 기반 갭 분석과 후속 질문 생성
- 결과물 저장 및 재사용 가능한 제품 구조

## MVP 범위
- 소스 입력
- 분석
- 후속 질문
- 지원동기/자기소개 초안, 경력기술서 초안, 면접 대비 리포트(핵심/심화) 생성
- 채용공고 맞춤 리라이트

## 산출물 라벨 원칙
- 현재 출력은 "완성본"이 아닌 "초안/도우미"로 표기한다.
- 권장 라벨:
  - 지원동기/자기소개 초안
  - 경력기술서 초안
  - 프로젝트 근거 보강 문구
  - 면접 대비 리포트 (핵심 질문/심화 질문)
